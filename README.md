# Community Detection in Social Media Using Social Networks Analysis

![Python](https://img.shields.io/badge/python-3.8+-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Community Detection and Social Network Analysis using graph-based algorithms, visualization techniques, and real-world network datasets.

---

## Project Overview

This project focuses on analyzing social network structures using graph-based techniques and community detection algorithms. The project includes graph visualization, community analysis, and comparison of different community detection methods on both real-world and synthetic datasets.

---

## Project Objectives

- Analyze social network structures
- Detect communities within graphs
- Visualize relationships between nodes
- Compare community detection algorithms
- Generate graph-based visualizations

---

## Datasets Used

| Dataset | Type |
|----------|------|
| Karate Club Network | Real-world |
| Dolphins Social Network | Real-world |
| Football Network | Real-world |
| Political Books Network | Real-world |
| Barabási–Albert Graph | Synthetic |
| Watts–Strogatz Graph | Synthetic |
| Erdős–Rényi Graph | Synthetic |

---

## Algorithms Implemented

- Louvain Community Detection
- Label Propagation
- Clique Detection

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/community-detection-social-network-analysis.git
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Open the Notebooks

```text
notebooks/01_community_detection_analysis.ipynb
notebooks/02_graph_visualizations.ipynb
```

### 4. Run All Cells

Execute all notebook cells to generate community detection and graph visualization outputs.
